# Test Document for ChatDoc

## Introduction

This is a test markdown document to verify that the ChatDoc application can process multiple document formats beyond PDF.

## Key Features

The system now supports:

- **PDF** - Portable Document Format
- **DOCX** - Microsoft Word documents
- **TXT** - Plain text files
- **MD** - Markdown files (like this one!)
- **HTML** - Web pages

## Technical Details

The backend uses:

- ChromaDB for vector storage
- Gemini 2.0 Flash for LLM responses
- Sentence Transformers for embeddings
- LangChain for text chunking
